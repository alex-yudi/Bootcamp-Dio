Este projeto é voltado para a prática de JS

Necessário abrir o console do navegador e "chamar a função" verificadorPalindromo(string)

Essa função vai receber o parâmetro "string" e verificar se ela é um palíndromo. Dando a resposta se a palavra em si é ou não é um.

Obs.: Necessário lembrar que, ao introduzir o parâmetro, deve-se colocar ele entre aspas.
