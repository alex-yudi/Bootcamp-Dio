Este projeto é voltado para a prática de JS

Necessário abrir o console do navegador e "chamar a função" comparadorDeNumeros(num1,num2)

Essa função vai somar os parâmetros num1 e num2. Fazendo o comparativo se eles são iguais, se a soma é maior que 10 e se a soma é maior que 20.
